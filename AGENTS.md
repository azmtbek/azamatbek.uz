# Agent instructions

This is a personal site (azamatbek.uz) built with TanStack Start + Vite.

## Package manager

Use **pnpm only**. Running `npm install` or `yarn` will throw an error from the `preinstall` hook.

```bash
pnpm install          # install deps
pnpm add <pkg>        # add a dependency
pnpm add -D <pkg>     # add a dev dependency
pnpm dlx <pkg>        # run a one-off binary (equivalent of npx)
```

## Dev workflow

```bash
pnpm dev      # start dev server (localhost:3000)
pnpm build    # production build
pnpm test     # run Vitest tests
pnpm check    # Biome lint + format check
```

## Key rules

- **Do not edit `src/routeTree.gen.ts`** — it is auto-generated by the Vite plugin on every save
- **Linting and formatting use Biome**, not ESLint or Prettier. Run `pnpm check` to verify
- **Tailwind CSS v4** — no `tailwind.config.ts`; theme customization goes inside `src/styles.css` using `@theme`
- **`app/` directory** at the project root is leftover Next.js scaffold — it is not active and should be ignored
- Routes go in `src/routes/`. The Vite plugin auto-discovers them and regenerates the route tree
- shadcn/ui components: `pnpm dlx shadcn@latest add <component>`
