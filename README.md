# azamatbek.uz

Personal site built with Next.js 16 App Router, React 19, Tailwind CSS v4, and shadcn/ui.

## Requirements

- **pnpm** (enforced — `npm install` and `yarn` will error)
- Node.js 20+

## Setup

```bash
pnpm install
```

## Commands

```bash
pnpm dev          # Start dev server at http://localhost:3000
pnpm build        # Production build (runs feed/sitemap generation first)
pnpm start        # Serve production build locally
pnpm test         # Run Vitest (server tests — requires dev server running)
pnpm test:build   # Build then run build smoke tests
pnpm lint         # Lint with Biome
pnpm format       # Format with Biome
pnpm check        # Lint + format together
```

## Tech stack

| Layer | Tool |
|---|---|
| Framework | [Next.js 16](https://nextjs.org/) (App Router) |
| Runtime | React 19 |
| Styling | [Tailwind CSS v4](https://tailwindcss.com/) |
| Components | [shadcn/ui](https://ui.shadcn.com/) |
| Linting | [Biome](https://biomejs.dev/) |
| Testing | [Vitest](https://vitest.dev/) |
| Package manager | pnpm |

## Project structure

```
src/
  app/                  # Next.js App Router pages
    layout.tsx          # Root layout (HTML shell, Header)
    page.tsx            # Home (/)
    blog/
      page.tsx          # Blog listing (/blog)
      [slug]/page.tsx   # Single post (/blog/:slug)
      tags/[tag]/page.tsx  # Tag filter (/blog/tags/:tag)
    projects/page.tsx   # /projects
    contact/page.tsx    # /contact
    not-found.tsx       # 404 page
    globals.css         # Global styles + Tailwind v4 theme
  components/
    Header.tsx          # Site nav ("use client" for active link detection)
    ui/                 # shadcn/ui components
  lib/
    posts.ts            # Markdown reading + frontmatter parsing
    utils.ts

public/
  posts/                # Markdown blog posts (*.md)
  rss.xml               # Generated by scripts/generate-feeds.ts
  sitemap.xml           # Generated by scripts/generate-feeds.ts

scripts/
  generate-feeds.ts     # Generates rss.xml + sitemap.xml into public/
```

## Adding a route

Create a `page.tsx` file inside `src/app/`:

```tsx
// src/app/about/page.tsx
import type { Metadata } from "next"

export const metadata: Metadata = { title: "About — Azamatbek" }

export default function AboutPage() {
  return <main>About</main>
}
```

## Adding shadcn/ui components

```bash
pnpm dlx shadcn@latest add button
```

## Adding a blog post

Create a Markdown file in `public/posts/`:

```markdown
---
title: "Post Title"
date: "2026-01-15"
description: "One-line summary shown in listing"
tags: ["tag1", "tag2"]
---

Post content here.
```

The slug is the filename without `.md`. Re-run `pnpm build` (or the dev server picks it up automatically).
